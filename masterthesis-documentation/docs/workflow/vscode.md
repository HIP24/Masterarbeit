# VSCode

| command               | Description                                                   |
|-----------------------|---------------------------------------------------------------|
| ^.*word.*$            | Delete lines containing word                                  |
| ^(\s)*$\n             | Delete blank lines                                            |
| 1(.*?)2  &rarr; 3$14  | replace 1 and 2 with 3 and 4, preserving the words in between |
